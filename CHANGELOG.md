# 更新日志

本文件记录 web-inputer 各版本的变更。

## [v0.1.1] - 2026-07-11

### 修复与改进

- 恢复键盘底行 `M` 右侧的退格键（⌫），支持按住连续删除；该键在 v0.1.0 之后的键盘重构中丢失。
- 屏幕预览窗比例改为跟随电脑屏幕实际比例自适应显示，竖屏显示器回退 16:10，不再固定 16:10 导致留黑边。
- 恢复预览窗标题显示。

## [v0.1.0] - 初始发布

- 首个发布版本：FastAPI 后端 + Vue 3 / Vite / naive-ui 前端。
- 屏幕预览、键鼠输入、触控板、文本输入、电源控制、窗口与桌面切换。
- PhotoSwipe 全屏查看屏幕截图。
